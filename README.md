# Flutter Interactive UI Demo

A simple Flutter application demonstrating the fundamentals of Flutter UI development using both **StatelessWidget** and **StatefulWidget**. The project showcases commonly used Flutter widgets along with basic state management through an interactive counter.


## 📖 Overview

This project is designed for beginners who want to understand how Flutter widgets work and how state is managed in a Flutter application.

It demonstrates the use of various widgets, layouts, asset images, and interactive buttons in a clean and organized user interface.


## ✨ Features

- StatelessWidget implementation
- StatefulWidget implementation
- AppBar
- Text Widget
- Icons
- Row Layout
- Column Layout
- Container Widget
- Stack Widget
- Asset Image
- Padding and SizedBox
- SingleChildScrollView
- Interactive Counter
- State Management using `setState()`
- Increment Counter
- Reset Counter


## 📱 Screens Included

The application contains the following UI sections:

- Application Header
- Introduction Text
- Icons Row
- Asset Image Display
- Container Widget Example
- Stack Widget Example
- Interactive Counter Section
